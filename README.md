# 🔬 Research Title Classifier

AI-powered classification of research paper titles.

## Features

- Classifies research titles into 5+ categories
- Real-time predictions with confidence scores
- Beautiful web interface
- Mobile responsive

## Categories

- Computer Vision
- Natural Language Processing (NLP)
- Reinforcement Learning
- Deep Learning
- Machine Learning

## Usage

1. Enter a research title
2. Click "Classify"
3. View prediction and confidence score

## Tech Stack

- Streamlit for UI
- TensorFlow/Keras for model
- scikit-learn for label encoding
